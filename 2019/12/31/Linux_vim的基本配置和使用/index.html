vim的基本配置和使用

2019-12-31
配置vim
cd ~    -->    vim .vimrc
    set nu        显示行号
    set ts=4    tab键等于4个空格键
    set autoindent    下面一行跟随上面一行的缩进而缩进
    set expandtab    
    set ruler    显示光标位置
    set nohls    代码不高亮
    syntax on    语法高亮
从命令行模式进入到插入模式
a    -->    进入插入模式并在光标之后进行添加。
i    -->    进入插入模式并在光标之前进行插入。
o    -->    进入插入模式并在当前（光标所在）行之下开启新的一行。
扩展模式下命令
:w    -->    将文件写入/存入磁盘
:q    -->    退出vi编译器
:wq    -->    将文件写入/存入磁盘并退出vi编译器
快速移动光标
在文件中的位置
    G    -->    光标移动到这个文件最后一行
    NG    -->    光标移动到这个文件第N行(N是数字)
    gg    -->    光标移动到这个文件第一行

在屏幕中的位置
    H    -->    光标移动到这个屏幕的最上方的那一行的第一个字符
    M    -->    光标移动到这个屏幕的中央的那一行的第一个字符
    L    -->    光标移动到这个屏幕的最下一行的那一行的第一个字符
删除
dd    -->    删除光标所在行
Ndd    -->    删除光标所在的向下N行(N是数字)
复制
yy    -->    复制光标所在的那一行
Nyy    -->    复制光标坐在的向下N行(N是数字)
粘贴
yy    -->    复制光标所在的那一行
Nyy    -->    复制光标坐在的向下N行(N是数字)
复原
u    -->    复原前一个操作
重做
Ctrl+c    -->    重做上一个操作
高级操作
vim -d 文件1 文件2    -->    同时打开两个文件来对比

vim 文件1 文件2 ..    -->    同时打开多个文件
    ls: 可查看当前打开多少个文件
    b 标号：切换操作文件
    sp/vs：将当前窗口水平/垂直方向拆分
    Ctrl+w：在命令模式下切换窗口
