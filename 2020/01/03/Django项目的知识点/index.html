Django常用的命令

创建一个项目：django-admin startproject 项目名
创建一个app：python manage.py startapp app名
运行项目：python manage.py runserver
进入django shell终端：python manage.py shell
生成迁移文件：python manage.py makemigrations pools
生成数据表：python manage.py migrate
创建管理员账号：python manage.py createsuperuser
修改密码：python manage.py changepassword

关于数据库

用Django开发一个项目我们应该给这个项目创建一个用户，并且只给予这个用户操作这个数据库的权限。

创建用户：
    create user 'yydafx'@'%' identified by '123456';

授予权限：
    部分权限：grant insert, delete, update, select, create, alter on django02.* to 'django02'@'%';
    所有权限：grant all privileges on django02.* to 'django02'@'%';

配置jet

现在Django的后台操作基本都是要jet

1)安装jet：
    pip install django-jet -i https://pypi.doubanio.com/simple

2)在INSTALLED_APPS的admin的前面注册jet：
    'jet'

3)执行迁移
    python manage.py migrate jet

4)URL配置
    from django.conf.urls import include

    path('jet/', include('jet.urls', 'jet')),

配置上传头像

1)新建文件夹
    在项目的根目录下新建media文件夹

2)setting配置：
    MEDIA_ROOT = os.path.join(BASE_DIR, 'media')
    MEDIA_URL = '/media/'

3)URL配置及写法
    from django.conf import settings
    from django.conf.urls.static import static

    urlpatterns = [
    path('admin/', admin.site.urls),
    ...
    ] + static(settings.MEDIA_URL, document_root=settings.MEDIA_ROOT)

    4)模板语言引入图片的写法
    {{ teacher.photo.url }}

Django随机验证码

1)在需要验证码路径的URL填写获取随机验证码陆游(/captcha/)
    <img id="code" src="/captcha/" height="33">

2)点击验证码图片切换下一张
    // 加载完页面执行箭头函数
    $(() => {
        $('#code').on('click', (evt) => {
            // 点击切换验证码
            $(evt.target).attr('src', '/captcha/?' + Math.random())
        })
    });

3)复制captcha.py和fonts(字体文件)到项目的app文件目录下

4)生成随机数(k值必须传入，否则会报错)
    一般会新建一个utils.py封装函数

    import random

    ALL_CHARS = '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ'

    def random_text(length=4):
        return ''.join(random.choices(ALL_CHARS, k=length))

5)views函数书写
    def get_captcha(request):
    """获取验证码图片"""
    code = random_text()
    image_data = Captcha.instance().generate(code)
    return HttpResponse(image_data, content_type='image/png')

对密码进行加密

一般会新建一个utils.py封装函数
import hashlib

def to_md5_hex(content):
"""生成hd5摘要"""
    return hashlib.md5(content.encode()).hexdigest()

在model.py的User类的里面：
# 重写save方法(对密码进行md5摘要)
def save(self):
    self.password = to_md5_hex(self.password)
    super().save()
