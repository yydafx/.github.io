Linux系统启停服务

启动：systemctl start nginx
停止：systemctl stop nginx
重启：system restart nginx
查看状态：systemctl status nginx
开机自启：systemctl enable nginx
禁止开机自启：systemctl disable nginx

常用的基本命令的整合及技巧

netstat -ntlp：查看当前运行的端口
jobs：查看在后台运行的命令
命令 &：将命令放到后台运行
fg %编号：把后台运行的命令放到前台运行
bg %编号：把前台运行的命令放到后台运行
Ctrl+z：将前台命令暂停并放到后台

!行号：执行该行号的命令
!v：执行最近一次以v开头的命令
echo  $PATH：查看配置的环境变量
onhup make >> ~/install.log &：将make命令在后台运行且不接受任何中断信号(可以杀死进程来结束)

取别名

vim .bushrc
    alias 别名='命令'
取消别名：
    unalias 别名
